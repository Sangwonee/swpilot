[개념 확인 질문 답변]

## 1. 스택 구조의 작동 방식

스택(Stack)은 데이터를 한쪽 끝에서만 넣고 빼는 자료구조이다. 가장 나중에 들어온 데이터가 가장 먼저 나가기 때문에 **후입선출(LIFO, Last In First Out)** 구조라고 한다.

예를 들어 데이터를 다음 순서로 넣으면:

```text
사과 → 바나나 → 체리
```

스택의 가장 위에는 마지막에 넣은 `체리`가 있다. 그래서 `pop()`을 하면 `체리`가 먼저 나온다.

## 2. 선입후출과 후입선출

정확히 말하면 스택은 **선입후출**이 아니라 **후입선출** 구조이다.

- 선입후출(FIFO): 먼저 들어온 데이터가 먼저 나감. 예: 큐(Queue)
- 후입선출(LIFO): 나중에 들어온 데이터가 먼저 나감. 예: 스택(Stack)

스택은 접시를 쌓는 모습과 비슷하다. 가장 먼저 놓은 접시는 아래에 있고, 가장 나중에 놓은 접시가 위에 있으므로 위의 접시부터 꺼내게 된다.

## 3. push(), pop(), peek(), isEmpty() 설명

- `push()`  
  스택의 가장 위에 데이터를 추가한다.

- `pop()`  
  스택의 가장 위에 있는 데이터를 꺼내고, 스택에서는 제거한다.

- `peek()`  
  스택의 가장 위에 있는 데이터를 확인만 한다. 데이터를 제거하지 않는다.

- `isEmpty()`  
  스택이 비어 있는지 확인한다. 비어 있으면 `true`, 데이터가 있으면 `false`를 반환한다.

## 4. 이 문제에서 구현한 스택 함수의 동작

`RoadToBiodome02.java`에서는 Java 기본 제공 `Stack`을 사용하지 않고 `CustomCharStack` 클래스를 직접 만들어 사용했다.

```java
private final char[] elements;
private int top;
```

- `elements`는 문자를 저장하는 배열이다.
- `top`은 스택의 가장 위 데이터가 저장된 위치를 나타내는 인덱스이다.
- 처음에는 아무 데이터도 없으므로 `top = -1`로 시작한다.

### push(char c)

```java
public void push(char c) {
    elements[++top] = c;
}
```

`++top`은 `top` 값을 먼저 1 증가시킨 뒤 그 위치에 문자를 저장한다. 즉, 새 문자를 스택의 맨 위에 추가한다.

### pop()

```java
public char pop() {
    return elements[top--];
}
```

`top--`은 현재 `top` 위치의 문자를 먼저 반환한 뒤, `top` 값을 1 감소시킨다. 즉, 스택의 맨 위 문자를 꺼내고 제거한 것처럼 만든다.

### isEmpty()

```java
public boolean isEmpty() {
    return top == -1;
}
```

`top`이 `-1`이면 스택에 저장된 데이터가 없다는 뜻이므로 `true`를 반환한다.

## 5. 문자열을 역순으로 만드는 과정

입력값이 `안녕하세요`라면 각 문자를 순서대로 스택에 넣는다.

```text
push('안')
push('녕')
push('하')
push('세')
push('요')
```

스택은 마지막에 들어온 값이 먼저 나오므로 `pop()`을 반복하면 다음 순서로 문자가 나온다.

```text
요 → 세 → 하 → 녕 → 안
```

따라서 최종 출력은 다음과 같다.

```text
요세하녕안
```

## 6. 예시 코드의 출력 결과

문제에 나온 코드는 다음과 같다.

```java
Stack<String> fruits = new Stack<>();
fruits.push("사과");
fruits.push("바나나");
fruits.push("체리");
System.out.println(fruits.pop());
System.out.println(fruits.peek());
```

실행 과정은 다음과 같다.

1. `사과`를 push한다.
2. `바나나`를 push한다.
3. `체리`를 push한다.
4. `pop()`은 가장 위에 있는 `체리`를 꺼내 출력한다.
5. 이제 가장 위에는 `바나나`가 있으므로 `peek()`은 `바나나`를 출력한다.

따라서 출력 결과는 다음과 같다.

```text
체리
바나나
```
