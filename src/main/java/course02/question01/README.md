수행과제

[최종 결과물]

여러 나무의 파동수가 적힌 배열 중 반복되지 않고 1번만 등장하는 숫자를 찾는 프로그램을구현한다.

[과제 목표]

Java의 배열 개념을 이해하고, 리스트를 활용해 데이터를 저장하고 접근할 수 있다.

특정 조건의 데이터를 찾기 위한 알고리즘을 구현할 수 있다.

[기능 요구사항]

사용자로부터 여러 나무의 파동 숫자가 기록된 값을 입력 받는다.

파동수의 최소값은 0이며 최대값은 1000이다.

주어진 배열에서 한 번만 등장하는 숫자를 찾아 출력한다.

입력값이 올바르지 않은 경우 사용자에게 안내 메시지를 출력한다.

[구현 지침]

파일의 이름은 RoadToBiodome01.java 이다.

리스트 변수를 사용하여 사용자의 입력 값을 배열 형태로 저장한다

배열 또는 리스트를 순회하면서 각 숫자의 등장 횟수를 파악하는 메서드를 구현한다.

main 함수 내에서 반복되지 않는 숫자를 찾아 출력한다.

입력값이 범위를 벗어나는 경우에 대한 예외 처리를 구현한다.

[개념 확인 질문 답변]

1. Java에서 배열과 리스트의 주요 차이점

배열은 생성할 때 크기가 고정되며, 한 번 정한 크기를 변경할 수 없다. 반면 리스트는 데이터를 추가하거나 삭제할 때 크기가 동적으로 변한다.

배열은 기본 자료형(int, double 등)과 참조 자료형(String, 객체 등)을 모두 저장할 수 있다. 리스트는 참조 자료형만 저장할 수 있으므로 int 같은 기본 자료형을 직접 저장하지 못하고, Integer와 같은 Wrapper 클래스를 사용해야 한다.

예를 들어 int 배열은 다음과 같이 사용할 수 있다.

```java
int[] numbers = new int[5];
```

리스트는 다음과 같이 사용할 수 있다.

```java
List<Integer> numbers = new ArrayList<>();
```

배열은 크기가 정해져 있고 인덱스로 빠르게 접근할 수 있어 고정된 개수의 데이터를 다룰 때 적합하다. 리스트는 add(), remove(), size() 같은 메서드를 제공하므로 데이터의 개수가 변할 수 있는 경우에 더 편리하다.

2. 구현한 프로그램에서 한 번만 등장하는 숫자를 찾기 위한 알고리즘 작동 방식

먼저 Command Line Arguments로 입력된 값을 문자열로 받은 뒤, 대괄호와 쉼표를 제거하고 숫자 단위로 나눈다. 각 값이 숫자인지 확인하고, 0 이상 1000 이하의 범위에 있는지 검사한 후 ArrayList에 저장한다.

그 다음 리스트의 각 숫자를 하나씩 확인한다. 현재 숫자가 리스트 안에서 몇 번 등장하는지 countOccurrences() 메서드로 센다. 이 메서드는 리스트 전체를 처음부터 끝까지 순회하면서 현재 숫자와 같은 값이 나올 때마다 개수를 증가시킨다.

findUniqueNumber() 메서드는 리스트의 앞에서부터 각 숫자의 등장 횟수를 확인하다가, 등장 횟수가 1인 숫자를 찾으면 그 값을 반환한다. main 함수에서는 반환된 값을 출력한다.

예를 들어 다음 입력이 있을 때:

```text
[5, 9, 16, 16, 5, 11, 5, 5, 9, 16, 16, 9, 9]
```

5는 4번, 9는 4번, 16은 4번 등장하고 11은 1번만 등장한다. 따라서 프로그램은 11을 출력한다.
