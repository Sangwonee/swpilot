수행과제

[최종 결과물]
숫자로 제공되는 식물들의 유전 정보를 기반으로, 서로 연관관계가 있는 식물 그룹의 개수를찾는 프로그램을 구현한다.

[과제 목표]
Depth First Search(DFS) 알고리즘을 이해하고, 재귀 함수 또는 스택을 이용해 주어진 문제를 해결할 수 있다.

[기능 요구사항]

식물간 연간관계를 입력받는다.

연구 대상 식물은 최대 100가지로 각 식물은 1~100 숫자로 표현된다.

연관관계가 있는 두 식물의 숫자는 쉼표로 구분되어 표현하며, 다음 연관관계는 띄어쓰기로 구분된다.

DFS 알고리즘으로 서로 연결된 식물 그룹을 찾는다.

연관관계가 있는 식물 그룹의 수를 출력한다.

예외 상황이 발생하는 경우 안내 메시지를 출력한다.

[구현 지침]

파일의 이름은 RoadToBiodome10.java 이다.

2차원 배열 형태를 이용해 식물간의 연관관계를 저장한다.

DFS 알고리즘 구현을 위해 재귀 함수 또는 Java 기본 제공 Stack을 사용한다.

main 함수에서 연결된 식물 그룹의 수를 출력한다.

아래의 예외 상황이 발생할 경우 안내 메시지를 출력하고 프로그램을 종료한다.

입력 값에 문자가 포함된 경우

식물의 범위를 벗어난 숫자가 포함된 경우

1,21 15,20 3,1

→ 2

(설명) : 문제 이해를 돕기 위한 설명으로 출력하지 않아도됨
⇒ 1번 식물과 21번 식물이 연관관계가 있음
⇒ 15번 식물과 20번 식물이 연관관계가 있음 → 15번 식물과 20번 식물은 하나의 그룹임
⇒ 3번 식물과 1번 식물이 연관관계가 있음 → 1번 식물, 21번 식물, 3번 식물은 하나의 그룹임
⇒ (15번 20번) (1번 21번 3번) 두개의 그룹이 발견됨

1,21 15,20 3,1 8,9 9,23 24,9 8,5

→ 3

4,5 7,3 90,2 13,4 8,2 1,61 16,21 56,7 38,4

→ 5



제약사항

사용자 입력은 Command Line Arguments 를 이용해 받는다.

Java 기본 제공 Set/Map 을 사용하지 않는다.

별도의 클래스나 파일을 생성하지 않는다.

보너스 과제

연관 관계가 있는 식물 그룹을 찾기 위해 BFS 알고리즘을 적용해 같은 결과가 나오는지확인한다.
