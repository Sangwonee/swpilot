# [개념 확인] HelloBiodome07 개념 이해 질문 & 답변 정리

## Q1. Java에서 문자를 다루는 데이터 타입 char와 String의 주요 차이점은 무엇인가요?
### **답변**
* **데이터 타입 분류**:
  * **`char`**: 자바의 8가지 **기본 자료형(Primitive Type)** 중 하나입니다.
  * **`String`**: 자바의 대표적인 **참조 자료형(Reference Type)** 클래스입니다.
* **저장하는 데이터의 차이**:
  * **`char`**: 단 **한 개의 문자**만 저장할 수 있으며, 홑따옴표(`'`)를 사용합니다. 내부적으로 문자의 유니코드 정수값을 직접 저장합니다.
  * **`String`**: 0개 이상의 문자들로 구성된 **문자열**을 저장할 수 있으며, 쌍따옴표(`"`)를 사용합니다.
* **메모리 저장 영역의 차이**:
  * **`char`**: **스택(Stack) 영역**에 문자(정수값)가 변수 공간 내에 직접 저장됩니다.
  * **`String`**: 실제 문자열 데이터는 **힙(Heap) 영역**에 저장되고, 스택 영역의 변수에는 그 데이터가 위치한 **메모리 주소값(참조값)**이 저장됩니다.
