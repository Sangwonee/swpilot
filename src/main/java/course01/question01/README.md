수행과제

[최종 결과물]

사용자의 이름을 입력 받아 이름+환영인사를 출력하는 Java 프로그램을 구현한다.

[과제 목표]

Java 프로그래밍을 할 수 있는 JDK 환경을 구축하고, Java 파일과 클래스 파일을 만들어 실행하는 방법을 숙지한다.

[기능 요구사항]

사용자로부터 이름을 입력 받는다.

입력받은 이름과 인사를 결합하여 사용자 맞춤 환영인사 를 출력한다.

사용자 이름에 띄어쓰기가 있는 경우 이름 전체가 출력되어야한다.

[구현 지침]

운영체제에 적합한 버전 11 이상 JDK를 설치한다.

Java 개발을 위해 널리 사용되는 IDE 중 하나를 선택해 설치한다.

파일의 이름은 HelloBiodome01.java 이다.

사용자로부터 입력 받은 이름을 저장할 변수 name 을 생성한다.

public static void main(String[] args) 메서드 내에서 인자를 받아 name 변수에 저장한다.

main 함수에서 사용자의 이름을 붙인 환영인사를 한 줄로 출력한다.

사용자가 이름을 1글자 이상 입력하지 않으면 재입력 안내 문구가 출력되고 프로그램이종료된다.

[결과 예시] 환영 문구는 다양하게 표현 가능

제약사항

사용자 입력은 Command Line Arguments 를 이용해 입력 받는다.

Scanner , BufferedReader 를 사용할 수 없다.

보너스 과제

사용자가 입력한 이름을 따옴표로 묶어서 출력한다

[Pass] 아돌프 블레인 → 반갑습니다, “아돌프 블레인”님

관리자의 이름 수가 항상 10글자를 넘는다고 가정하고, 10번째 이후 문자를 잘라서 출력한다

String 클래스의 length() , substring() 메서드를 사용할 수 없다.

반복문(for, while)을 사용할 수 없다.
