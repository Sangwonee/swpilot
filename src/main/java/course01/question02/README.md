수행과제

[최종 결과물]
입력받은 태양광, 풍력 그리고 지열 에너지 생산량을 각각 숫자로 변환한 뒤, 총 에너지 생산량의 합계를 출력하는 프로그램을 구현한다.

[과제 목표]

Java의 다양한 숫자 데이터 타입을 이해하고, 목적에 맞게 선택할 수 있다.

Java의 캐스팅(Casting) 개념을 이해하고, 명시적 형변환과 자동 형변환을 사용할 수있다.

[기능 요구사항]

세 종류 에너지 생산량을 입력 받는다.

에너지 생산량은 정수이며, 마이너스 값이 될 수 없다.

각 에너지 생산량의 최대값은 30,000이다.

모든 에너지 생산량을 합한 총 에너지 사용량을 출력한다.

[구현 지침]

파일의 이름은 HelloBiodome02.java 이다.

태양광, 풍력 그리고 지열 에너지 생산량을 저장할 숫자 변수 세 개를 생성한다.

입력 받은 에너지 생산량을 각 변수에 형변환하여 저장한다.

문자열을 숫자로 변환하기 위해서는 Integer.parseInt() 또는 Double.parseDouble() 메서드를 사용할 수 있다.

main 함수에서 변환된 숫자들의 총 합을 계산한 결과를 출력한다.

숫자가 아닌 값을 입력 받으면 에러 메시지를 출력하고 프로그램을 종료한다.

[결과 예시]

10000 20000 30000
-> 총 에너지 사용량은 60000입니다.

제약사항

사용자 입력은 Command Line Arguments 를 이용해 입력 받는다.

보너스 과제

세 가지 에너지가 전체 에너지 생산량 중 어느 정도 비율을 차지하는지 계산하여 출력한다.

결과는 퍼센트로(%)로 나타낸다.

10000 20000 30000
-> 총 에너지 사용량은 60000입니다.
태양광 16.666666666%, 풍력 33.33333333%, 지열 50.0000000%

[개념 확인 질문]

Java에서 사용할 수 있는 정수 숫자 데이터 타입들에 대해 설명하고, 각 변수의 데이터
타입을 선택한 이유를 질문한다.

byte, short, int, long 각 데이터 타입의 범위와 에너지 생산량의 조건을 고려한 답변

Java의 자동 형변환과 명시적 형변환에 대해 질문한다.

범위가 큰 데이터 타입과 범위가 작은 데이터 타입 간의 변환 차이 설명

만약 대규모의 데이터를 저장해야 한다면, int 와 short 둘 중 어떤 데이터 타입을 선택하는 것이 메모리 사용량 측면에서 더 효율적인지 질문한다.

각 데이터 타입의 바이트 크기와 형변환 처리 단계 등을 고려한 답변
