## Q1. 사용자가 입력한 값의 유효성을 검사하는 이유와 코드에서 유효성 검사를 구현한 방법에 대해 설명해 주세요.

## Q2. 맨 아래줄의 기둥과 같이 나무 정중앙에 문자를 출력하는 정렬 로직은 어떻게 구현했나요?
### **답변**
* **정중앙 좌표의 산술 조건**:
  * 나무의 총 높이가 $H$일 때, 나무의 가장 밑바닥 줄(마지막 줄)이 갖는 최대 너비는 $2H - 1$이 됩니다.
  * 따라서 기둥과 매 행의 중앙 문자가 정확히 수직 정렬을 이루기 위해 필요한 **왼쪽 공백의 개수는 언제나 $H - 1$개**로 고정됩니다.
* **중앙 정렬 구현 로직**:
  * **기둥 출력**: 기둥 `'|'` 기호는 나무의 최하단 중앙에 위치해야 하므로, 루프를 통해 **$H - 1$ 개의 공백**을 먼저 연이어 출력한 후 `'|'` 기호를 출력하여 기둥의 정렬을 완벽하게 일치시켰습니다.
  * **매 줄(Row) 출력**: $i$번째 줄(0부터 시작)을 출력할 때, 공백의 수는 줄이 내려갈수록 줄어들고 별의 수는 늘어납니다. 따라서 **$H - 1 - i$ 개의 공백**을 출력 ➔ **$i$ 개의 왼쪽 별** 출력 ➔ **정중앙 문자(1개)** 출력 ➔ **$i$ 개의 오른쪽 별**을 출력하는 대칭 구조를 활용해 항상 정중앙에 수직축이 유지되도록 설계하였습니다.
